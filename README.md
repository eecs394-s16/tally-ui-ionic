# tally-ui-ionic

##Environment Setup:

1. install node.js

2. install ionic

`npm install -g cordova ionic`

##Run the app:

  `ionic serve`
  
or

  `ionic run ios`

##Limitations
- Masonry drag and drop functionality (not free form)
- No currency conversions (parses price and shows with $ sign but doesn't actually convert)
